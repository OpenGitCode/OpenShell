# 🚀 OpenShell Roadmap & Vision

Developed by **OpenGit**, OpenShell is evolving from a powerful CLI to a distributed intelligence network. This roadmap outlines the strategic pillars that will define the future of the platform.

---

## 🧠 Pillar 1: Autodidaxis (Continuous Learning)
*Making OpenShell intuitive by learning from every interaction.*

- [x] **Core Memory System**: Persistent storage for user preferences (`memory.json`).
- [ ] **Preference Extraction**: Automatically identify and store user preferences from chat history (e.g., preferred languages, output formats).
- [ ] **Contextual Priming**: Use learned data to pre-configure agents for specific tasks without user input.
- [ ] **Behavioral Adaptation**: Adjust agent personality and verbosity based on user feedback.

## 🧩 Pillar 2: Smart Orchestration
*Moving from single-task execution to complex workflow management.*

- [ ] **TaskFlow Auto-Generator**: Automatically suggest and create TaskFlow sequences based on repeated patterns.
- [ ] **Cross-Skill Communication**: Allow skills to share state and data directly without user mediation.
- [ ] **Intelligent Triage**: Automatically route requests to the most capable model or skill based on complexity.
- [ ] **Parallel Execution**: Run non-dependent tool calls in parallel to boost performance.

## 🌐 Pillar 3: Distributed Connectivity
*Expanding OpenShell beyond a single machine into a unified network.*

- [ ] **ShellHub Integration**: Native support for remote management via ShellHub.
- [ ] **Node Discovery**: Auto-discovery of OpenShell nodes in a local network (Raspberry Pi, Servers, Containers).
- [ ] **Remote Skill Execution**: Invoke a skill on a remote node as if it were local.
- [ ] **Secure Node Handshake**: Zero-config encrypted connection between OpenShell instances.

---

## 📈 Release Strategy

- **v2.1.x**: Hardening, Security, and Gemini Integration.
- **v2.2.0**: The "Memory Update" (Full Autodidaxis implementation).
- **v2.5.0**: The "Orchestration Update" (TaskFlow automation).
- **v3.0.0**: The "Distributed Network" (Multi-node support).
