# Skill: Summarize Pro

This skill allows for researching topics and returning ultra-compact summaries.

## Critical Rules
1. **DO NOT USE MARKDOWN**: The response must be plain text. No bold, headers, or code blocks unless strictly necessary.
2. **MAXIMUM BREVITY**: If you can say it in 10 words, don't use 20.
3. **STRUCTURE**: Use simple bullets (-) if needed, but avoid long paragraphs.

## Example Response
Topic: Mars Weather
Response: Extreme cold, CO2 atmosphere, global dust storms, -60°C average.
